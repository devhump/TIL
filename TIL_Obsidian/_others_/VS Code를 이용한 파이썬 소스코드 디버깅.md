### VS code 디버깅 관련 단축키
|     단축키     |             기능             |
|:--------------:|:----------------------------:|
|      `F9`      |         break point          |
|      `F5`      |         디버깅 실행          |
| `ctrl` + `F5`  |       디버깅 없이 실행       |
|     `F10`      |    인터프리터 한 줄 이동     |
|     `F11`      |        함수 안 디버깅        |
| `shift` + `F5` | 디버깅 모드 종료 (실행 취소) |
