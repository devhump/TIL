#단축키, #shortcut 

### Mac OS

| `shift`+ `cmd` + `.`         | 폴더(파인더) 내 숨김파일 보이기 |
| ---------------------------- | ------------------------------- |
| `command` +`ctrl`+`space`    | 이모지                          |
| `shift` + `command(⌘)` + `3` | 전체 화면 캡쳐                  |
| `shift` + `command(⌘)` + `4` | 부분 지정 캡쳐                  |



### VS Code for mac

|                                  |             |
| -------------------------------- | ----------- |
| `shift` + `i`  or `option` + `i` | insert 이슈 |
