## 수업중 유용히 쓰이는 단축키 모음 (초초 요약 ver. 22/07/08)

### 유튜브 

| `t`           | 영화관 모드 설정     |
| ------------- | -------------------- |
| `f`           | 전체화면 설정 / 해제 |
| `shift` + `>` | x 배속 (+0.25)       |



### 윈도우

| `win` + ▶  또는 `win` + ◀        | 화면 좌/우로 밀착             |
| -------------------------------- | ----------------------------- |
| `tab` + `alt` 또는 `tab` + `win` | 실행중인 프로그램 한눈에 보기 |
| `win` + `.`                      | 이모지                        |
| `win` + `shift` + `s`            | 스크린샷 (윈도 자체 기능)     |



### Typora

| ctrl + t     | 테이블 생성                |
| ------------ | -------------------------- |
| ctrl + enter | 마크다운 코드블럭에서 탈출 |



### Git-bash

| `shift` + `insert` | 붙여넣기(git-bash 환경 내) |
| ------------------ | -------------------------- |
|                    |                            |
